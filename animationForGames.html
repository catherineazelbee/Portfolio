Animation For Games: Work In Progress

Animation by Catherine Azelby

Made using Autodesk Maya 2025

Character models sourced from Mixamo and Professor Bill Stout

Rigging done with Advanced Skeleton

Project Goals

Upon completion, this project will include a walk cycle, hammer weight cycle, run cycle and a movement tree. These animations will then be imported into Unity where we will work on controlling these animations with input from a game controller. Expected completion is December 2024.

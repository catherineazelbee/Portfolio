Character Animation Demo

Animation For Games: Character Animation Project

Animation by Catherine Azelby

Made using Autodesk Maya 2025

Character Model from Kiel Figgins

Project Description

This project is a current work-in-progress, with expected completion on December 13, 2024. The goal of this project was to create animation for a pre-rigged character for a video game. We were to include a walk and run cycles, as well as an idle and special move animation. I'm a big fan of Marvel's Deadpool so I used the free-to-use Deadpool rig by Kiel Figgins to complete this project.
